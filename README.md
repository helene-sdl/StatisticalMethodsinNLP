# NLP Statistics Project

**Author:** Helene Seidl (MLT, helene.seidl@univie.ac.at)

NLP and statistical analysis of a Project Gutenberg text (*Charles Seignobos*, ID: 29637), written as an R Sweave (`.Rnw`) document.

## Requirements

R packages: `gutenbergr`, `tidytext`, `dplyr`, `ggplot2`  


## Contents

1. Text download & deduplication
2. Word counts & bigrams (Zipf's Law)
3. Bigram dependence — chi-square test on *"according to"*
4. Hypothesis testing explained (courtroom analogy)
5. Character-level entropy across 1,000-word segments
6. Entropy explained
7. 95% confidence interval for mean entropy (t-test)
8. Confidence intervals explained
9. Naïve Bayes classification across 4 text sections
10. TF-IDF analysis of distinctive words/bigrams per section
